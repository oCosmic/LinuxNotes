# LinuxNotes
Notes from me going through the Linux Bible:

Sequential commands are used by separating them with semicolons 

Piping between commands are used with |. 

**Creating and using aliases: you can effectively create a shortcut to any command and options that you want to run later. **
Ex: alias p='pwd' ; ls -CF'
alias rm= ' rm -i'
P is assigned to run the command pwd and then run ls -CF
The second example runs the rm command with the -i option each tim eyou rm.
